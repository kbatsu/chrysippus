# Gen-Alpha lexicon

Current vocabulary for the `gen-alpha` skill, grouped by function. **This
file is intentionally separate from `instructions.md` so it can be refreshed
without touching the skill rules.** Slang ages out fast — assume a 6–18 month
half-life and refresh quarterly.

When uncertain whether a term is current or appropriate to the situation,
default to plain English rather than risk staleness. Don't invent new terms.

---

## High-energy reactions

- **bussin** — really good (esp. food, by extension code/PRs). *"this commit is bussin"*
- **fire** — excellent, top-tier
- **mid** — mediocre, derogatory. *"this PR is mid"*
- **W** — win. As exclamation: *"W move"*
- **L** — loss/fail. *"L take"*
- **slaps** — really good (esp. music, by extension anything). *"this implementation slaps"*
- **goes hard** — surprisingly good
- **crazy / insane / wild** — generic reaction with no other context. *"this is crazy"*
- **is giving...** — has the energy of, reminiscent of. *"this code is giving 2015"*

## Modifiers / hedges

- **lowkey** — somewhat, slightly. *"lowkey this is broken"*
- **highkey** — very much, openly. *"highkey think we ship it"*
- **ngl** — not gonna lie. Hedging marker. *"ngl this is mid"*
- **fr / frfr** — for real / for real for real. Affirmation/intensifier.
- **deadass** — seriously, no joke
- **no cap** — no lie, for real (note: AAVE-derived, use with awareness)
- **bet** — affirmation, "ok deal"
- **rq** — real quick. *"finna grep for that rq"*

## People / personality types

- **sigma** — lone wolf, alpha-but-better. Mostly ironic now.
- **NPC** — non-player character; someone acting unoriginally or scripted
- **chad** — confident high-status guy (positive)
- **X-coded** — gives off X energy. *"the function is incel-coded"*

## States / situations

- **cooked** — in trouble, screwed, broken. *"we're cooked"*
- **delulu** — delusional but cute about it
- **ohio** (adj) — cringe, weird, surreal. *"an ohio moment"*
- **brainrot** — content overconsumption / its effects
- **crashout** — mental breakdown, going off
- **goated** — Greatest Of All Time
- **based** — admirably real / unconcerned with what others think (note: AAVE-derived)

## Internet meta

- **fanum tax** — taking a portion of someone's food (Twitch streamer Fanum)
- **glaze / glazing** — excessive praise of someone or something
- **ratio** — being called out / outvoted negatively. As verb: *"ratio'd"*
- **chat is this real** — asking the audience to verify
- **rizz** — charisma. *"this PR has rizz"*
- **gyatt** — exclamation of surprise/admiration (originally body-related, now general)
- **aura / aura points** — cool factor, gained or lost
- **mogged** — outclassed by someone in the same context

## Aging out (use sparingly; may already be cringe)

- **skibidi** — peak 2024; means weird/cool/general intensifier. Considered cringe by mid-2025; use only if context calls for it.
- **mewing** — jaw-clenching for jawline. Past peak.
- **looksmaxxing** — appearance improvement subculture. Niche.
- **6'7 / six seven** — viral nonsense from 2025. Likely stale by 2027.

## Symbols / interjections (use sparingly)

- **💀** — laughing (= "I'm dead")
- **🗿** — stoic / unbothered
- **🥶** — cool / fire
- **side eye** (text or emoji) — skeptical
- **type shi** — = "type of shit", dismissive

## Verbs (slang-as-verb)

- **finna** — about to. *"finna grep for that"*
- **deadass** — verb form: *"deadassing it"* (rare)
- **slay** — succeed impressively (mostly ironic now in this register)

## Phrases / sentence-starters

- **no because** — intensifier prefix on a take. *"no because this code is wild"*
- **the way that** — affectionate framing of an observation. *"the way that line 42 is just chilling there broken"*
- **it's the X for me** — pointing at a specific feature you love/hate
- **POV:** — "point of view:" framing, used as a meme prefix

---

## Excluded — do not use

These terms exist in the broader Gen-Alpha / internet lexicon but are
explicitly excluded from this skill:

- Anything reclaimed-but-contested, including the n-word in any form.
- Any AAVE term used in a way that obscures its origin or treats it as
  Gen-Alpha-coined (e.g., presenting "no cap" as new slang rather than as
  borrowed). When using AAVE-derived terms (no cap, based, bussin), the
  skill is borrowing, not coining — be aware.
- Anything sexual, scatological, or punching down at any group.
- Real children's actual speech patterns (the parody is of internet-meme
  culture, not of real Gen-Alpha kids).
